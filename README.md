# P01 - Python Fundamentals for ISC

Este proyecto demuestra los conceptos fundamentales de programación en Python, diseñado para estudiantes de ISC (Ingeniería en Sistemas Computacionales).

## Descripción

P01 es un programa educativo que cubre los conceptos básicos de Python, incluyendo:

- **Tipos de datos**: enteros, decimales, cadenas, booleanos y listas
- **Operaciones básicas**: aritméticas y lógicas
- **Estructuras de control**: condicionales (if/else) y bucles (for/while)
- **Funciones**: definición y uso de funciones con parámetros y valores de retorno
- **Interacción con el usuario**: entrada y salida de datos
- **Manejo de errores**: try/except básico

## Requisitos

- Python 3.6 o superior

## Uso

Para ejecutar el programa:

```bash
python main.py
```

o

```bash
python3 main.py
```

## Conceptos Demostrados

### 1. Tipos de Datos
El programa muestra ejemplos de:
- Números enteros (`int`)
- Números decimales (`float`)
- Cadenas de texto (`str`)
- Valores booleanos (`bool`)
- Listas (`list`)

### 2. Operaciones Básicas
Incluye operaciones aritméticas como suma, resta, multiplicación, división, etc.

### 3. Estructuras de Control
- Condicionales para tomar decisiones
- Bucles para repetir operaciones

### 4. Funciones
Ejemplos de funciones que calculan áreas geométricas y demuestran el paso de parámetros.

### 5. Interacción con Usuario
Sección interactiva que solicita datos al usuario y los procesa.
